# Habit Tracker App

The Habit Tracker App is a command-line interface (CLI) application that helps users create, manage, and analyze their daily and weekly habits. It provides features to add new habits, mark them as completed, and analyze progress over time using Python.

## Features

- **Create and Manage Habits**: Add, edit, and delete daily or weekly habits.
- **Track Habit Completion**: Mark habits as completed for specific dates.
- **Analyze Habits**: View habit statistics, streaks, and identify most successful habits.
- **Persistent Storage**: Store data in JSON format for example data and SQLite/MySQL database for real tracking.
- **Modular Design**: Easily extendable and maintainable code with well-defined modules.
- **Unit Testing**: Comprehensive unit tests using `pytest` to ensure application stability.

## Technologies Used

- **Python**: Version 3.7+
- **CLI Framework**: `click`
- **Database**: SQLite or MySQL for data persistence
- **Data Storage**: JSON for example data and initial loading/testing
- **Testing**: `pytest` for unit tests
